# 看一眼 / SwiftGlance

菜单栏/任务栏系统监控。macOS 用菜单栏托盘图标画两行读数；Windows/Linux 用一个置顶小窗复刻同款两行版式。

## Language

**悬浮窗**：
Windows/Linux 上复刻 macOS 菜单栏两行读数的无边框置顶小窗（Tauri `widget` 窗口）。macOS 不用它，改用菜单栏托盘图标。
_Avoid_: 小窗、浮窗、tooltip、widget（中文行文里统一叫「悬浮窗」）

**锚点**：
悬浮窗**首次显示**的默认位置 = Windows 通知区「显示隐藏图标」`^` 箭头的左缘（紧贴其左侧，留 2px 间隙）。用户一旦拖动，就改用拖动后的位置、不再回锚点。
_Avoid_: 默认位置、初始坐标

**读数**：
悬浮窗/托盘图标里显示的两行监控文本（`row1`/`row2`，如 `c97%m77%` / `↓66B↑ 0B`），由三平台共用的 `format_rows()` 生成，保证逐字符对齐。
_Avoid_: 数据、指标文本（「指标」指后台采集的 `Metrics`，「读数」指其渲染后的文本）
