# Triage 标签

这些 skill 使用五个标准的 triage 角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 我们 tracker 中的标签 | 含义                       |
| -------------------------- | --------------------- | -------------------------- |
| `needs-triage`             | `needs-triage`        | 维护者需要评估这个 issue   |
| `needs-info`               | `needs-info`          | 等待报告者补充更多信息     |
| `ready-for-agent`          | `ready-for-agent`     | 已完整说明，AFK agent 可接手 |
| `ready-for-human`          | `ready-for-human`     | 需要人类实现               |
| `wontfix`                  | `wontfix`             | 不予处理                   |

当某个 skill 提到某个角色（例如「打上 AFK-ready triage 标签」）时，使用本表中对应的标签字符串。

如需改用你实际使用的标签词汇，编辑右侧那一列即可。
